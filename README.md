# dva-gui
A gui tool for dva.

> Under Development

## Development

Install Dependencies
```
$ git clone github/repo/of/dva-gui
$ cd dva-gui
$ npm run install
$ cd web
$ npm install
$ cd ../app
$ npm install
```

Run
```
$ cd web
$ npm run dev
$ cd ../app
$ npm run dev
```

## Build
```
$ cd web
$ npm run build
$ cd ../app
$ npm run build
$ cd ..
$ npm run package
```
