# dva-gui

A gui tool for dva.

> Under development.

## Getting Started

Install dependencies and run build in watch mode.

```bash
$ npm install
$ npm run build -- --watch
```

Then clone dva-ast and npm link.

```bash
$ git clone github/repo/of/dva-ast
$ npm install
$ cd path/to/dva-gui
$ npm link path/to/dva-ast
```

Finally, open another terminal tab and run:

```bash
$ npm start
```
