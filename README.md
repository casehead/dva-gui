# Dva-GUI

Dva-GUI is a static analysis GUI tool for dva projects. It gives you a clear picture of your application's data flows.

## Dev

Install dependencies.

```bash
$ npm install
```

Start webpack and watch.

```bash
$ npm run dev
```

Then open another terminal, and run start.

```bash
$ npm start
```

## Build

```bash
$ npm run pack
```
